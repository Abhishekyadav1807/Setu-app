3. LEGAL PROCEDURE GUIDE (Module 2)
   ✅ Listing page showing 5 procedure cards with icons and timeline badges
   ✅ Data fetched from backend API with fallback data
   ✅ Info banner noting BNS/BNSS 2023 legal references

   ✅ Procedure Detail Page (for each of the 5 procedures):
      - Purpose section with full description
      - Quick info cards (Fees, Timeline, Official Website)
      - Required Documents – checklist with tick marks
      - Step-by-Step Process – vertical timeline with numbered steps
      - Important Tips – highlighted tip cards
      - Back navigation

   ✅ Procedures covered:
      a) FIR Filing
         - 6 required documents listed
         - 6-step process (Visit station → Meet SHO → Narrate → Sign → Get copy → Follow up)
         - Fees: Free
         - Timeline: Immediate
         - 6 tips including Zero FIR and Section 175(3) BNSS
         - Official link: digitalpolice.gov.in

      b) RTI Application
         - 5 required documents listed
         - 6-step process (Identify authority → Draft → Pay ₹10 → Submit → Track → Appeal)
         - Fees: ₹10 per application
         - Timeline: 30 days (48 hours for life/liberty)
         - 6 tips
         - Official link: rtionline.gov.in

      c) Consumer Complaint
         - 8 required documents listed
         - 6-step process (Contact seller → Legal notice → Register e-Daakhil → File → Pay fee → Attend)
         - Fees: Free for claims up to ₹5 lakh
         - Timeline: 3-5 months
         - 6 tips
         - Official link: edaakhil.nic.in

      d) Marriage Registration
         - 10 required documents listed
         - 6-step process (Collect docs → Apply → 30-day notice → Appear → Sign → Collect certificate)
         - Fees: ₹100-500
         - Timeline: 1-30 days
         - 6 tips
         - Official link: india.gov.in

      e) Passport Application
         - 7 required documents listed
         - 6-step process (Register → Fill form → Pay → Book appointment → Visit PSK → Police verification)
         - Fees: ₹1,500 normal / ₹3,500 tatkal
         - Timeline: 30-45 days normal / 1-3 days tatkal
         - 6 tips
         - Official link: passportindia.gov.in

11. MONGODB DATABASE (Mongoose)
    ✅ MongoDB connection with mongoose (mongodb://127.0.0.1:27017/setu-platform)
    ✅ 4 Mongoose models:

    a) Law Model
       - slug (unique), title, category, icon
       - overview (text)
       - keySections (array: section, title, description)
       - examples (array: title, scenario, outcome)
       - useCases (array: title, description)
       - importantLinks (array: label, url)
       - timestamps

    b) Procedure Model
       - slug (unique), title, icon
       - purpose (text)
       - requiredDocuments (string array)
       - steps (array: stepNumber, title, description)
       - fees, timeline, officialLink
       - tips (string array)
       - timestamps

    c) Scheme Model
       - name, slug (unique), category
       - eligibility, benefits, description
       - requiredDocuments (string array)
       - officialWebsite
       - timestamps

    d) Document Model
       - type (enum: fir-draft, rti-application, legal-notice,
              consumer-complaint, rent-notice)
       - formData (mixed/flexible)
       - generatedAt (date)
       - timestamps

12. API ENDPOINTS
    ✅ GET  /api/laws                – Get all laws (summary fields)
    ✅ GET  /api/laws/:slug          – Get full law by slug
    ✅ GET  /api/procedures          – Get all procedures (summary)
    ✅ GET  /api/procedures/:slug    – Get full procedure by slug
    ✅ GET  /api/schemes             – Get schemes (with ?search= and ?category= query filters)
    ✅ GET  /api/schemes/categories  – Get distinct scheme categories
    ✅ GET  /api/schemes/:slug       – Get single scheme by slug
    ✅ POST /api/documents/generate  – Save generated document to database
    ✅ GET  /api/documents           – Get all generated documents (latest 50)
    ✅ GET  /api/health              – Server health check
